Como usar

  • Baixe balloon.js e chame-o de sua página

  • Baixe balloon.css e chame-o da mesma página

  • A página deve chamar pelos métodos:
    • Balloon.success(mensagem,opcoes) ou Balloon['success'](mensagem,opcoes)
    • Balloon.info(mensagem,opcoes) ou Balloon['info'](mensagem,opcoes)
    • Balloon.warning(mensagem,opcoes) ou Balloon['warning'](mensagem,opcoes)
    • Balloon.danger(mensagem,opcoes) ou Balloon['danger'](mensagem,opcoes)
    • Balloon.primary(mensagem,opcoes) ou Balloon['primary'](mensagem,opcoes)
    • Balloon.balloon(mensagem,opcoes) ou Balloon['balloon'](mensagem,opcoes)
    • Balloon.inflate(mensagem,tipo_personalizado,opcoes) ou Balloon['inflate'](mensagem, tipo_personalizado,opcoes)

    Obs.: O parâmetro "tipo_personalizado" adicionará a classe balloon-tipo_personalizado para o alerta de acordo com o tipo especificado. Para adicionar uma imagem para o tipo personalizado deve-se passar a url da imagem na propriedade "opcoes.imageUrl".
    Obs.2: O parâmetro "opcoes" é sempre opcional.

  • Adicione suas "opcoes" personalizadas no parâmetro "opcoes", ou subscreva o objeto "Balloon.default_options" para alterar as opções globais

    Obs.: o parâmetro "opcoes" sempre tem prioridade sob o objeto "Balloon.default_options"
    Obs.2: Em "opcoes" só é necessário anunciar os atributos desejados e com o mesmo nome que foram anunciados no objeto de opções padrão("Balloon.default_options")

Balloon

Alertas JavaScript

Balloon é um código javascript que permite criar alertas com facilidade para o seu projeto, onde podem ser emitidos erros, notificações e outras respostas/feedbacks para o usuário
